# ArcadeLevels

Levels:
1. Pong
2. Breakout
3. Tetris
4. Pacman
5. Pinball
6. Space Invaders
7. Bubble Bobble
8. Defender/Zaxxon
9. Centipede
10. 2D Platformer
11. 3D Pacman
12. 3D Space Invaders
13. "Dino Finder"
14. 3D FP Platformer
15. Hard Minesweeper
